Visão computacional

Definição

A visão Computacional tem por finalidade obter, a partir de uma ou várias imagens, informações geométricas, topológicas ou físicas sobre os dados que as originaram.

Gomes & Velho, 2008

Demo

Nesse experimento, é utilizado a jsc3d para renderização de modelos 3d a partir de formatos conhecidos(.obj, .3ds, entre outros); js-objectdetect para detecção de padrões de objetos; canvas para renderização em 3d e Web API na captura de imagens da câmera.
